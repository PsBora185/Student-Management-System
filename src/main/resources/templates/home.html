About This Project

The Student Management System is designed to simplify student record management. You can easily view student details, add new students, update existing records, and delete entries when necessary.

Built using Spring Boot, Thymeleaf, and Bootstrap 5, following MVC best practices for a clean, maintainable structure.

  • User-friendly interface
  • Full CRUD functionality
  • Responsive design
  • Scalable MVC structure
